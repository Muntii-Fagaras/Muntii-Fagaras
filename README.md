## 動作環境

- Windows 10/11
- Linux 5.0　以降

## 使い方

1. gui-baseを起動します。
2. UI上のアイコンを触ってみたりしてください!!

## 特徴

- クロスプラットフォーム
- 最新のC++20の採用
